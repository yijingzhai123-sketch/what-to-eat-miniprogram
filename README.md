# "今天吃什么"微信小程序
🍃 清新自然风格，解决每日吃饭选择困难症

## ✨ 功能特点
1. **随机选菜**：一键帮你选择今天吃什么，带动画滚动效果
2. **菜品管理**：自主上传菜品图片、名称、分类标签、备注
3. **标签分类**：支持预设标签+自定义标签，快速筛选菜品
4. **今日记录**：自动记录每日选择，支持换一个
5. **历史记录**：查看最近吃过的菜品，避免重复
6. **本地存储**：所有数据存在手机本地，无需服务器，隐私安全
7. **清新UI**：莫兰迪绿色系自然风格，使用舒适

## 📱 使用方法
1. 下载安装"微信开发者工具"
2. 选择"导入项目"，选择本文件夹（what-to-eat-miniprogram）
3. AppID可以选择"测试号"（不需要注册小程序）
4. 点击"预览"，扫码即可在iOS微信上打开使用
5. 如果要发布到自己的小程序账号，可以替换为自己的AppID后提交审核

## 📂 目录结构
```
what-to-eat-miniprogram/
├── app.js              # 全局逻辑
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── sitemap.json        # 站点配置
```

## 🎨 设计风格
- 主色调：薄荷绿系 `#88c9a3` / `#5ab47e`
- 背景色：浅嫩绿 `#f0f9f4`
- 圆角、柔和阴影，无生硬边框，整体清新治愈
- 预置了6个默认菜品，首次打开即可体验，无需先加菜
